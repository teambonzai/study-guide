# Key Terms & Glossary — Control of Microorganisms

All definitions are drawn from the professor's slides (Bauman Ch. 9 / Cowan Ch. 11). Plain-language
phrasing is mine; the science is the deck's.

## The core vocabulary (Table 9.1 — learn these cold)

| Term | Definition |
|---|---|
| **Sterilization** | Destruction/removal of **ALL** microorganisms and viruses in or on an object |
| **Disinfection** | Destruction of **most** microbes and viruses on a **nonliving** surface (leaves endospores) |
| **Antisepsis** | Reducing the number of microbes on **living tissue** (e.g., skin) |
| **Degerming** | **Mechanical** removal of microbes from a surface (hand washing, alcohol swab) |
| **Sanitization** | Reducing microbes to **safe public-health levels** (scalding dishwater) |
| **Pasteurization** | Using **heat** to kill pathogens & reduce spoilage organisms in foods/beverages |
| **Aseptic** | An environment or procedure **free of contaminants** |
| **Decontamination** | The **mechanical removal** of most microbes |
| **-cide / -cidal** | Suffix: the agent **kills** (bactericide, fungicide, virucide) |
| **-stasis / -static** | Suffix: the agent **inhibits** growth without killing (bacteriostatic) |
| **Sterilant** | An agent that eliminates **all** microbes **including spores** |
| **Disinfectant** | Kills fungi, bacteria, and viruses on surfaces (not spores) |
| **Sanitizer** | Reduces the **number** of bacteria to safe levels |
| **Antiseptic** | A germicide safe enough to use on **living tissue** |
| **Germicide** | Any agent that kills microbes; rated **high / intermediate / low** level |

## Physical-control terms

| Term | Definition |
|---|---|
| **Physical control** | A physical force (heat, radiation, cold, pressure) is exerted on microbes |
| **Microbicidal** | Kills microbes (generally **higher** temperatures) |
| **Microbistatic / microbiostatic** | Inhibits microbial growth (generally **lower** temperatures, drying) |
| **Moist heat** | Heat carried by steam/water; kills faster & at lower temps than dry heat |
| **Dry heat** | Hot air / flame; needs higher temps & longer times; hits endotoxin |
| **Autoclave** | Steam **under pressure** — the preferred sterilizer; **15 psi → 121 °C**, ~15 min |
| **Thermal Death Time (TDT)** | Shortest time to kill all test microbes **at a set temperature** |
| **Thermal Death Point (TDP)** | Lowest temperature that kills all microbes in a sample **in 10 min** |
| **Pasteurization (flash)** | 71.6 °C for 15 seconds |
| **Pasteurization (batch)** | 63–66 °C for 30 minutes |
| **Thermoduric microbes** | Heat-tolerant microbes that survive pasteurization |
| **Boiling** | Disinfection (not sterilization); ~30 min; leaves endospores |
| **Hot-air oven** | Dry-heat sterilizer; 150–180 °C for 2–4 h; for heat-resistant items |
| **Incineration** | Burning/flaming — destroys everything completely |
| **Endospore** | A dormant, ultra-resistant bacterial form; **the hardest thing to kill** |
| **Endotoxin** | The lipopolysaccharide of gram-negative walls; destroyed by prolonged dry heat |

## Radiation & low-temperature terms

| Term | Definition |
|---|---|
| **Radiation** | Energy from atomic activity moving at high velocity through matter/space |
| **Ionizing radiation** | Gamma rays, X-rays, cathode rays — **deeply penetrating**, destroys DNA, **sterilizes** |
| **Nonionizing radiation** | **UV** — shallow, forms thymine dimers in DNA, **disinfects** |
| **Ultraviolet (UV)** | ~100–400 nm; germicidal lamp at **254 nm**; surface/air/water disinfection |
| **Germicidal lamp** | A UV lamp (254 nm) used to disinfect air, surfaces, and water |
| **Refrigeration** | Slows metabolism; **halts** growth of most pathogens but doesn't kill |
| **Psychrophile** | A cold-loving microbe that can still multiply in refrigerated food |
| **Desiccation** | Drying; removes water; **microbiostatic only** |
| **Lyophilization** | Freeze-drying; long-term culture preservation without ice-crystal damage |
| **Osmotic pressure** | High salt/sugar → hypertonic → cells lose water and desiccate |
| **Hypertonic** | A solution concentrated enough to pull water out of a cell |

## Chemical-control terms

| Term | Definition |
|---|---|
| **Chemical control** | Chemicals used to kill or inhibit microbes |
| **Tincture** | An agent dissolved in alcohol (or a water–alcohol mix) |
| **Phenol (carbolic acid)** | The original surgical antiseptic; denatures proteins, disrupts membranes |
| **Phenolics** | Milder phenol derivatives (Lysol, triclosan); active around organic matter |
| **Alcohols** | Intermediate-level; denature proteins; 70% ethanol skin swab; evaporate fast |
| **Halogens** | Iodine, chlorine, bromine, fluorine; oxidize/denature enzymes (Betadine, bleach) |
| **Iodophor** | Iodine complexed with a carrier (e.g., **Betadine**) |
| **Chlorhexidine** | Chlorine + two phenolic rings; bactericidal (gram +/–), not sporicidal; mild, rapid |
| **Oxidizing agents** | Peroxides, ozone, peracetic acid; oxidize enzymes; **high-level** |
| **Hydrogen peroxide** | Disinfects/sterilizes surfaces; **catalase neutralizes it → not for open wounds** |
| **Peracetic acid** | An effective **sporocide** used to sterilize equipment |
| **Surfactant** | "Surface-active" chemical that lowers surface tension (soaps, detergents, quats) |
| **Soap** | Good **degerming** agent (lifts microbes off) but **not antimicrobial** |
| **Quaternary ammonium compounds (quats)** | Positively charged detergents; **low-level** disinfectants; human-safe |
| **Heavy metals** | Silver, copper, mercury, zinc; alter protein 3-D shape; low-level |
| **Silver nitrate (1%)** | Prevents gonococcal (N. gonorrhoeae) blindness in newborns |
| **Thimerosal** | A mercury preservative used to preserve vaccines |
| **Aldehydes** | Glutaraldehyde & formaldehyde; cross-link/denature proteins & nucleic acids |
| **Glutaraldehyde** | Disinfects (short exposure) and sterilizes (long exposure) |
| **Formalin** | Formaldehyde solution used in embalming and room/instrument disinfection |
| **Gaseous agents** | Ethylene oxide, propylene oxide, β-propiolactone; closed-chamber sterilization |
| **Ethylene oxide** | A gas sterilant for heat-/moisture-sensitive items; explosive, carcinogenic |
| **Antimicrobials** | Antibiotics + synthetics used mainly to treat disease |
| **Phenol coefficient** | An agent's killing power expressed as a ratio to phenol (>1.0 = better) |

## Mechanical-control terms

| Term | Definition |
|---|---|
| **Mechanical control** | Physically **removing** microbes from the environment (not killing them) |
| **Filtration** | Straining fluid/air through pores too small for microbes to pass |
| **Membrane filter** | Thin cellulose-acetate/polycarbonate sheet with a controlled pore size |
| **0.22 µm filter** | The standard filter for sterilizing heat-sensitive liquids (traps most bacteria) |
| **HEPA filter** | High-Efficiency Particulate Air filter; removes airborne microbes (cabin air, OR) |
| **21 CFR 211.46** | The regulation governing ventilation/air filtration in drug manufacturing |
| **Directional airflow** | Controlled air movement/dilution that limits internal spread of bacteria |

## Acronyms & abbreviations

| Acronym | Full name | In one line |
|---|---|---|
| **TDT** | Thermal Death Time | Shortest kill-all time at a set temperature |
| **TDP** | Thermal Death Point | Lowest kill-all temperature (in 10 min) |
| **UV** | Ultraviolet | Nonionizing radiation; surface/air disinfection at 254 nm |
| **psi** | Pounds per square inch | Pressure unit; autoclave runs at 15 psi |
| **UHT** | Ultra-High Temperature | Very hot, very brief milk treatment (can sterilize) |
| **HEPA** | High-Efficiency Particulate Air | Air filter that removes airborne microbes |
| **Quats** | Quaternary ammonium compounds | Low-level surfactant disinfectants |
| **EtO / EO** | Ethylene oxide | Gas sterilant for heat-sensitive equipment |
| **CFR** | Code of Federal Regulations | U.S. regulations (21 CFR 211.46 = air control) |
