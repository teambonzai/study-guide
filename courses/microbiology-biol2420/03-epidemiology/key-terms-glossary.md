# Key Terms & Acronyms — Epidemiology

All definitions are drawn from your professor's slides and lecture. Plain-language phrasing is
mine; the science is his.

## Acronyms (decode these instantly)

| Acronym | Full name | In one line |
|---|---|---|
| **R₀** | Basic reproduction number | New cases one case makes in a fully susceptible population (contagiousness) |
| **CDC** | Centers for Disease Control and Prevention | US agency (Atlanta); publishes the weekly Morbidity & Mortality Report |
| **WHO** | World Health Organization | Coordinates public health internationally |
| **STI / STD** | Sexually Transmitted Infection / Disease | Reported at the end of the month |
| **TB** | Tuberculosis | Chronic infectious disease; endemic/pandemic; report same-day |
| **HPV** | Human Papillomavirus | The "confounded" STI example (Gardasil-era over-testing) |
| **UP** | Universal Precautions | Treat every patient specimen as potentially infectious |
| **BSI** | Body Substance Isolation | Techniques used within Universal Precautions |
| **CBC** | Complete Blood Count | Lab test; a differential can hint at infection type |

## Core epidemiology terms

| Term | Definition |
|---|---|
| **Epidemiology** | Study of the distribution and prevalence of disease in a population |
| **Morbidity** | The number of cases (the pathological state seen across a population) |
| **Mortality** | The number of deaths from a disease in a population |
| **Prevalence** | The **total** number of cases in a population at a given time |
| **Incidence** | The number of **new** cases — tells you how fast a disease is spreading |
| **Reproductive number (R₀)** | Expected new cases from one case where all are susceptible; higher = more contagious |
| **Confounding** | A distortion that makes data misleading (e.g., over-testing inflates a count) |
| **Baseline** | The normal/expected level of a disease, used for comparison to detect an outbreak |
| **Etiology** | The study of the **cause** of a disease |
| **Koch's postulates** | Classic criteria linking a microbe to a disease (limited: unculturable microbes, cofactors, ethics) |
| **Patient zero (initial case)** | The first case in an outbreak, found by clustering cases and tracing back |
| **Contact tracing** | Interviewing cases about place, people, and time to find the source |
| **Surveillance** | Collecting, analyzing, and reporting disease data (mostly passive; ~a year behind) |
| **Reportable (notifiable) disease** | A disease that by law must be reported to authorities |

## Patterns of disease in a population

| Term | Definition |
|---|---|
| **Sporadic** | A few scattered, unrelated cases — no real population effect |
| **Epidemic** | A short-term increase in cases in a given population |
| **Endemic** | An agent present in a population/location all the time (e.g., *Coccidioides* in the SW US) |
| **Pandemic** | A worldwide increase — cases on 6 continents (e.g., COVID-19, HIV/AIDS, TB) |
| **Outbreak** | A localized epidemic (often handled at local/state level) |

## Transmission & spread

| Term | Definition |
|---|---|
| **Communicable disease** | Can be transmitted host-to-host (contagious); includes mother→fetus (vertical) |
| **Contagious** | Highly communicable |
| **Non-communicable disease** | Acquired but not passed person-to-person (e.g., Lyme disease) |
| **Contact transmission** | Direct (sex), indirect (handshake, object), or droplet (sneeze) |
| **Vehicle transmission** | Spread by air (droplet nuclei, aerosols), water, or food |
| **Vector transmission** | Spread by a live animal — biological or mechanical |
| **Biological vector** | Actively participates in the pathogen's life cycle; takes a blood meal (mosquito → malaria) |
| **Mechanical vector** | Carries the agent on its body without being infected (housefly) |
| **Fomite** | An inanimate object that can transmit an agent (doorknob, phone) |
| **Portal of entry / exit** | How a pathogen gets into / out of the body (often the same route; usually secretions/excretions) |
| **Oral–fecal route** | Transmission via fecal contamination of food/water/hands |
| **Vertical transmission** | Passage from mother to developing fetus |
| **Zoonosis / zoonoses** | Infection indigenous to animals but naturally transmissible to humans (≥150; humans don't sustain it) |

## Reservoirs & carriers

| Term | Definition |
|---|---|
| **Reservoir** | The primary natural habitat a pathogen originates from (living or nonliving) |
| **Source** | The specific individual/object from which an infection is actually acquired |
| **Carrier** | A person who harbors and spreads an agent, often with little/no symptoms |
| **Asymptomatic carrier** | Harbors and spreads the agent with no symptoms |
| **Nonliving reservoir** | Soil or water that harbors a pathogen (e.g., Katrina floodwater) |

## Stages, signs & outcomes

| Term | Definition |
|---|---|
| **Incubation period** | From exposure until clinical symptoms appear (you can be contagious here) |
| **Prodromal period** | The earliest, vague symptoms |
| **Illness** | Full signs and symptoms (usually most infectious) |
| **Decline** | Symptoms ease as the body wins |
| **Convalescence** | Recovery (may still be infectious) |
| **Symptom** | Subjective evidence of disease felt/reported by the patient |
| **Sign** | Objective evidence measured by a practitioner (BP, temperature) |
| **Syndrome** | A group of signs + symptoms that together define a disease |
| **Asymptomatic / subclinical** | No symptoms, but signs of infection may still be present |
| **Impairment** | A lasting outcome — cognitive or physical — even after the agent is gone |

## Pathogen & host factors

| Term | Definition |
|---|---|
| **Pathogenicity** | A pathogen's ability to cause disease |
| **Adherence** | The pathogen's ability to attach to and stay on the host |
| **Invasiveness** | How easily a pathogen enters the body |
| **Antigenic variation** | Mutation that changes a pathogen's surface, evading immunity/treatment |
| **Intracellular pathogen** | Lives inside host cells to evade defenses (HIV, malaria, chlamydia, rickettsia) |
| **Infectious dose** | Number of organisms needed to cause infection (Shigella ~10 ≪ Salmonella ~100,000) |
| **Toxin** | A poison a pathogen produces (e.g., botulism) |
| **Endospore / cyst / capsule** | Survival structures pathogens use to persist and resist defenses |
| **Host susceptibility** | How likely a host is to get infected (age, health, immunization, genetics, culture) |
| **Immunocompromised / immunosuppressed** | Weakened immunity (AIDS, chemo, transplant drugs) → more susceptible |

## Control & clinical

| Term | Definition |
|---|---|
| **Quarantine** | Isolating exposed people/animals to limit spread |
| **Pasteurization** | Heating to kill pathogens (60 °C/30 min; flash 72 °C/15 s) |
| **Immunization** | Best population-level protection — immunize as many people as possible |
| **Nosocomial infection** | An infection acquired during a hospital stay (2–4 million/yr in the US) |
| **Exogenous infection** | A nosocomial infection acquired from the healthcare environment |
| **Endogenous infection** | A nosocomial infection arising from the patient's own microbiota |
| **Iatrogenic infection** | An infection resulting from a medical procedure |
| **Universal Precautions (UPs)** | Assume all patient specimens are infectious; use BSI techniques |
| **Medical asepsis** | Practices (esp. hand washing) that reduce infection transmission |
| **Prophylactic antibiotics** | Antibiotics given to prevent infection (e.g., after surgery) |
