# Key Terms & Acronyms — Immune Response

All definitions are drawn from your professor's slides and lecture. Plain-language phrasing is
mine; the science is his.

## Acronyms (decode these instantly)

| Acronym | Full name | In one line |
|---|---|---|
| **Ag** | Antigen | Anything foreign that triggers an immune response (= non-self) |
| **Ab** | Antibody | A protein that binds and inactivates an antigen (= immunoglobulin) |
| **Ig** | Immunoglobulin | Another word for antibody (Ig**M**, Ig**A**, Ig**D**, Ig**G**, Ig**E**) |
| **MHC** | Major Histocompatibility Complex | Genes coding the "self ID" markers on cells |
| **APC** | Antigen-Presenting Cell | A cell (MHC Class II) that shows a piece of an invader to the T4 helper |
| **CD4 / T4** | Cluster of Differentiation 4 | The **T-helper** cell — the "on switch" of immunity (same as "T4 helper") |
| **CD8** | Cluster of Differentiation 8 | Cytotoxic/cytolytic T cell marker |
| **TCR** | T-Cell Receptor | The receptor a T cell uses to read an APC's display |
| **NET** | Neutrophil Extracellular Trap | A web neutrophils cast to trap and kill microbes |
| **MAC** | Membrane Attack Complex | Complement proteins that punch pores in a pathogen's membrane |
| **ADCC** | Antibody-Dependent Cell-mediated Cytotoxicity | Killing big parasites from the outside |
| **NK** | Natural Killer (cell) | Kills tumor and virus-infected cells; an APC |
| **WBC** | White Blood Cell | Leukocyte — the defensive blood cells |
| **RBC** | Red Blood Cell | Erythrocyte — **the only cell with no "self" MHC markers** |
| **HIV / AIDS** | Human Immunodeficiency Virus / Acquired Immunodeficiency Syndrome | Attacks CD4 helpers → cripples immunity |

## Key terms

| Term | Definition |
|---|---|
| **Immunology** | Study of the body's 2nd and 3rd lines of defense |
| **Self** | Cells belonging to your body; carry self markers (all cells except RBCs) |
| **Non-self** | Anything foreign → an **antigen** |
| **Immunocompetent** | Having a healthy, working immune system |
| **Immunocompromised** | Weakened immune system (e.g., AIDS, chemotherapy) |
| **Antigen (Ag)** | Foreign substance that elicits an immune response |
| **Antibody (Ab)** | Y-shaped protein that binds antigens; made by plasma cells |
| **Epitope** | The specific surface marker on an antigen that an antibody binds |
| **Immunoglobulin (Ig)** | = antibody |
| **Titer** | Lab measure of antibody amount; higher = more/recent immunity |
| **MHC Class I** | Genes coding **self markers** on every nucleated cell |
| **MHC Class II** | Markers on **APCs** that present antigen to T4 helpers |
| **B lymphocyte (B cell)** | Matures in **bone marrow**; makes antibodies (humoral immunity) |
| **T lymphocyte (T cell)** | Matures in the **thymus**; runs cell-mediated immunity |
| **T helper (T4/CD4)** | Sentinel; the only cell that **turns on** the immune response (recognition) |
| **T cytolytic (cytotoxic)** | Destroys infected cells (inactivation) |
| **T suppressor** | Regulates/modifies the response (regulation) |
| **Plasma cell** | A B-cell product that secretes antibodies |
| **Humoral immunity** | Antibody-mediated (B cells); fights **extracellular** pathogens |
| **Cell-mediated immunity** | T cell–driven; fights **intracellular** pathogens, tumors, grafts |
| **Cytokine** | Signaling molecule that recruits/activates more immune cells |
| **Phagocytosis** | A cell engulfing and digesting a microbe (steps: **CAKE**) |
| **Phagosome** | Vesicle holding the ingested microbe |
| **Lysosome** | Vesicle of digestive enzymes |
| **Phagolysosome** | Phagosome + lysosome fused → microbe destroyed |
| **Chemotaxis** | Movement toward a chemical signal (step 1 of phagocytosis) |
| **Diapedesis** | WBCs squeezing out of blood vessels into tissues |
| **Opsonization** | Coating a microbe so phagocytes grab it more easily |
| **Agglutination** | Antibodies clumping microbes together for easier destruction |
| **Neutralization** | Antibodies blocking a virus/toxin from acting |
| **Granulocyte** | WBC with granules: neutrophil, eosinophil, basophil |
| **Agranulocyte** | WBC without granules: lymphocyte, monocyte/macrophage |
| **Neutrophil** | First responder to **bacteria**; makes NETs |
| **Eosinophil** | Responds to **parasites & allergies**; attacks big worms externally |
| **Basophil / Mast cell** | Release histamine in allergic/inflammatory reactions |
| **Macrophage / Monocyte** | Large phagocyte; also an APC |
| **Helminth** | Parasitic worm |
| **Eosinophilia** | Elevated eosinophils → sign of worm infestation |
| **Complement** | Cascade of serum proteins that lyse microbes, neutralize toxins/viruses, enhance phagocytosis |
| **Membrane Attack Complex (MAC)** | Complement pores that burst a pathogen |
| **NETs** | Neutrophil webs that trap microbes (non-phagocytic killing) |
| **Perforin** | Enzyme that punches holes in a parasite's surface |
| **Lytic enzyme** | Enzyme that breaks down a pathogen's surface |
| **Primary response** | First exposure; IgM first, then IgG rises and stays |
| **Secondary (anamnestic) response** | Re-exposure; fast, huge IgG burst from memory |
| **Anamnestic** | "Memory" response on re-exposure |
| **Immune** | Re-infection will be milder/shorter — not "never infected again" |
| **Immunoprivileged site** | Tissue graftable without rejection: upper skin, lens, cornea |
| **Anaphylaxis** | Severe IgE allergic reaction → constriction → hypoxia; treat with EpiPen |
| **Side effect (of a vaccine)** | Normal signs the immune system is working (soreness, mild fever) |
| **Adverse reaction** | Abnormal harmful response (hives, anaphylaxis, blood clots) — different from a side effect |
| **Acquired immunity** | Immunity built through exposure; needs recognition, specificity, and memory |
| **Active immunity** | Your body makes its own antibodies (infection or vaccine) — long-term |
| **Passive immunity** | You receive ready-made antibodies (mom, gamma globulin) — short-term |
| **Natural active** | Immunity from catching the infection → IgG for life |
| **Natural passive** | Mom's antibodies via placenta + colostrum/breast milk (short-term) |
| **Artificial active** | Immunity from a vaccine (long-term) |
| **Artificial passive** | Ready-made antibodies given as treatment (short-term, ~6 wks–3 mo) |
| **Gamma globulin** | Pooled, concentrated IgG from thousands of donors; passive immunity; works best regionally |
| **Convalescent plasma** | IgG-rich plasma from recovered patients (e.g., COVID) given for passive immunity |
| **Antivenom / antitoxin (antisera)** | IgG raised in sheep/goats against a venom/toxin, then given to a patient |
| **Attenuated** | A microbe **weakened but still alive** (e.g., Sabin polio); can revert to virulence |
| **Inactivated** | A microbe that is **killed/inactive** (e.g., Salk polio); safe, can't revert |
| **Reversion to virulence** | When an attenuated (live) vaccine mutates back to a disease-causing form |
| **Salk vaccine** | The **inactivated** polio vaccine |
| **Sabin vaccine** | The **live-attenuated** polio vaccine |
| **Vaccine platforms (5)** | mRNA/DNA · live-attenuated · inactivated · subunit · viral-vector |
| **Subunit vaccine** | Uses only a piece of the microbe (e.g., a protein/flagella) |
| **Viral-vector vaccine** | A harmless virus (e.g., adenovirus) delivers the genetic instructions |
| **mRNA vaccine** | Nucleic acid tells your cells to make a protein you then make antibodies against |
| **Adjuvant** | An additive (usually a salt) that irritates locally → recruits WBCs → stronger response |
| **Herd immunity** | ~70–80% of a population immune stops spread, protecting the unvaccinated |
| **Thimerosal** | Mercury preservative that stabilizes proteins in multi-dose vials; no link to autism |
| **Nosode** | Dangerous folk "immunization" using live, virulent body secretions — not a vaccine |
