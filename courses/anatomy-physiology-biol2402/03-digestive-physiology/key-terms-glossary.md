# Key Terms & Acronyms — Digestive Physiology

All definitions are drawn from your professor's slides and lecture. Plain-language phrasing is
mine; the science is his.

## Acronyms & abbreviations

| Term | Full name | In one line |
|---|---|---|
| **GI tract** | Gastrointestinal tract | The continuous tube, mouth → anus (= alimentary canal) |
| **HCl** | Hydrochloric acid | Stomach acid; denatures proteins, activates enzymes |
| **LES** | Lower esophageal sphincter | Cardiac sphincter; gates esophagus → stomach; fails → GERD |
| **GERD** | Gastroesophageal reflux disease | Acid refluxes into the esophagus → heartburn → Barrett's |
| **ACh** | Acetylcholine | Vagus/parasympathetic secretagogue of HCl (M3 receptor) |
| **PPI** | Proton pump inhibitor | Omeprazole/pantoprazole — blocks the H⁺/K⁺ pump |
| **CCK** | Cholecystokinin | Intestinal hormone; targets stomach, pancreas, gallbladder |
| **GIP** | Gastric inhibitory peptide | Intestinal hormone released by carbohydrates |
| **GLP-1** | Glucagon-like peptide-1 | Slows gastric emptying + raises insulin (Ozempic/Wegovy) |
| **NaHCO₃** | Sodium bicarbonate | Alkaline pancreatic juice that buffers acidic chyme |
| **SGLT / GLUT** | Sodium-glucose transporter / glucose transporter | Carry monosaccharides across the intestinal cell |
| **SIBO** | Small intestinal bacterial overgrowth | Colon bacteria in the ileum → malabsorption |
| **NSAID** | Non-steroidal anti-inflammatory drug | e.g., aspirin; reduces protective mucus → ulcers |

## Structures & anatomy

| Term | Definition |
|---|---|
| **Alimentary canal** | Another name for the GI tract (mouth → anus, ~9 m) |
| **Accessory organs** | Help digestion but aren't part of the tube: salivary glands, teeth, tongue, liver, gallbladder, pancreas |
| **Mucosa / submucosa / muscularis externa / serosa** | The four layers of the GI-tract wall (inner → outer); the mucosa is the histological focal point |
| **Parotid gland** | Largest salivary gland; watery, enzyme-rich saliva; dark-staining serous cells |
| **Sublingual gland** | Salivary gland under the tongue; mucus-rich saliva; pale translucent mucus cells |
| **Submandibular gland** | Salivary gland producing mixed (enzyme-rich + mucous) saliva |
| **Bolus** | Ball of chewed food + saliva that is swallowed |
| **Esophageal hiatus** | Hole in the diaphragm the esophagus passes through |
| **Rugae (gastric folds)** | Stomach folds that let it stretch/expand |
| **Pyloric sphincter** | Muscle gating chyme from stomach → duodenum |
| **Gastric pits/glands** | Stomach mucosa invaginations lined by mucus, parietal, chief, and G cells |
| **Islets of Langerhans** | Endocrine pancreas; secrete insulin/glucagon into blood |
| **Acinar cells** | Exocrine pancreas cells that secrete digestive enzymes |
| **Lobule / central vein / portal triad** | Liver microanatomy — hexagonal unit / its central drain / the corner trio (portal vein, hepatic artery, bile duct) |
| **Sinusoid** | Fenestrated liver capillary lined by Kupffer cells |
| **Bile canaliculi** | Tiny channels that collect bile from hepatocytes → bile ducts |
| **Common bile duct / cystic duct** | Carry bile from the liver/gallbladder toward the duodenum |
| **Hepatopancreatic ampulla (of Vater)** | Where the common bile duct and main pancreatic duct join before entering the duodenum |
| **Sphincter of Oddi** | Muscle gating the ampulla; controls bile/pancreatic-juice release |
| **Major duodenal papilla** | The opening where the ampulla empties into the duodenum |
| **Duodenum / jejunum / ileum** | The three regions of the small intestine (proximal → distal) |
| **Plicae circulares / villi / microvilli** | Nested surface-area amplifiers of the small intestine |
| **Lacteal** | Lymphatic vessel inside a villus; absorbs fats (chylomicrons) |
| **Cecum / appendix / colon / rectum / anal canal** | The regions of the large intestine |

## Cells & molecules

| Term | Definition |
|---|---|
| **Mucus (neck) cells** | Secrete bicarbonate-rich alkaline mucus that protects the stomach lining |
| **Parietal cells** | Secrete hydrochloric acid (HCl) via the H⁺/K⁺ pump |
| **Chief cells** | Secrete pepsinogen (→ pepsin in HCl) |
| **G cells** | Enteroendocrine cells that release gastrin |
| **D cells** | Release somatostatin — the primary inhibitor of acid |
| **S cells** | Release secretin in response to acid |
| **Enteroendocrine cells** | Intestinal cells that release hormones (secretin, CCK, GIP, GLP-1) |
| **Kupffer cells** | Liver macrophages that clear pathogens/debris from portal blood |
| **Pepsin / pepsinogen** | Protein-digesting enzyme / its inactive zymogen |
| **Secretagogue** | A substance that triggers another cell to secrete (ACh, gastrin, histamine for HCl) |
| **Gastrin** | G-cell hormone → more HCl from parietal cells |
| **Somatostatin** | D-cell hormone; primary brake on acid secretion |
| **Zymogen** | An inactive enzyme (name ends in "-ogen," e.g., trypsinogen, pepsinogen) |
| **Enteropeptidase** | Brush-border enzyme that converts trypsinogen → trypsin (starts the cascade) |
| **Bile / bile salts** | Liver secretion that emulsifies fat into small droplets |
| **Micelle** | Bile-salt package of fatty acids/monoglycerides for absorption |
| **Chylomicron** | Repackaged absorbed fat that exits via the lacteal (lymph) |

## Processes & clinical

| Term | Definition |
|---|---|
| **Peristalsis** | Adjacent smooth-muscle layers contracting rhythmically to move material A → B |
| **Churning** | Stomach's mechanical digestion (3 muscle layers) that makes chyme |
| **Chyme** | Acidic slurry of food + HCl + enzymes leaving the stomach |
| **Gastric emptying** | Metered release of chyme through the pyloric sphincter |
| **Denaturation** | Acid unfolding a protein so enzymes can reach its bonds |
| **Emulsification** | Bile breaking large fat droplets into small ones (chemical digestion of fat) |
| **Alkaline tide** | Blood leaving the stomach turns alkaline (HCO₃⁻ in as H⁺ pumps out) |
| **Cephalic / gastric / intestinal phase** | The three phases of digestion (anticipatory / food-in-stomach / chyme-in-intestine) |
| **GERD → Barrett's esophagus** | Chronic reflux erodes the esophageal lining; Barrett's is the precursor to esophageal cancer |
| **Gastric ulcer** | HCl-eroded hole in the stomach lining from too little mucus (H. pylori, stress, NSAIDs) |
| **Pyloric stenosis** | Narrowed pyloric sphincter (common in newborns) → stomach can't empty |
| **Metabolic alkalosis / acidosis** | Excess vomiting (lose acid) / diarrhea (lose base) |
| **Gut microbiome** | Resident colon bacteria aiding digestion, immunity, and metabolism |
| **Defecation reflex** | Rectal stretch → spinal PSNS reflex → internal sphincter relaxes; external sphincter is voluntary |
